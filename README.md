# YouTube Clone - Enhanced Version

A fully functional YouTube clone built with HTML, CSS, and JavaScript featuring modern UI/UX, dark mode, video player, comments, and more!

## 🚀 Features

### **Core Functionality**
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Dark Mode**: Toggle between light and dark themes with persistent storage
- **Video Player Modal**: Full-screen video player with controls
- **Search Functionality**: Real-time search with voice search support
- **Category Filtering**: Filter videos by Music, Gaming, Cooking, Technology, Sports
- **Dynamic Video Loading**: Load more videos with smooth animations

### **Interactive Features**
- **Video Actions**: Like, dislike, share, and save videos
- **Comments System**: Post and view comments on videos
- **Channel Subscription**: Subscribe/unsubscribe to channels
- **Notifications Panel**: View notifications with unread indicators
- **Upload System**: Video upload modal with drag-and-drop interface

### **Navigation & UI**
- **Smart Sidebar**: Collapsible navigation with categories and sections
- **Header Controls**: Search, upload, notifications, and user menu
- **Filter Buttons**: Quick filtering of video content
- **Toast Notifications**: User feedback for actions
- **Smooth Animations**: Modern transitions and hover effects

### **Content Management**
- **Multiple Sections**: Home, Trending, Subscriptions, Library, History
- **Video Categories**: Organized content by type
- **Dynamic Content**: Programmatically generated video data
- **Responsive Grid**: Adaptive video grid layout

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and accessibility
- **CSS3**: Flexbox, Grid, CSS Variables, Animations
- **JavaScript (ES6+)**: DOM manipulation, event handling, local storage
- **Font Awesome**: Comprehensive icon library
- **Google Fonts**: Roboto typography
- **CSS Custom Properties**: Dynamic theming system

## 🚀 How to Use

1. **Clone/Download**: Get the project files
2. **Open in Browser**: Launch `yt.html` in any modern browser
3. **Explore Features**:
   - Toggle dark mode with the moon/sun icon
   - Search for videos using the search bar
   - Filter videos by category buttons
   - Click any video to open the player modal
   - Try the upload feature
   - Check notifications
   - Post comments on videos
